Location Scout

An app that helps photographers save locations along with photos and notes
Location Scout is an Android app that helps photographers and filmmakers save their scouted locations.
Features:
  • Save a location with a title, notes, tags, and photos
  • View your saved locations in a list, grid or map
  • Search your saved locations by text or tags
  • Sync your saved locations across any number of Android devices using a Google Drive account
  • Light and dark themes

Location Scout is currently in beta but anyone can join. Join the beta!

App requires Android 4.0+

Join the beta

Location Scout is currently in beta but anyone can join! Here's how:

1. Join the Google+ community

Only members of the Location Scout beta test Google+ community can install the app. The community is also used to report bugs and suggest new features!

2. Become a tester

Before you can install the app, you have to become a beta tester. This allows you to install and receives updates for the beta version of the app.

3. Install the app

Now you should be able to download the Location Scout app on the Play store.
